# estimation-system

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Install dependencies for the server
```
npm install
```

### Run the client & server concurrently
```
npm run dev
```

### Run the express server only
```
npm start
```
